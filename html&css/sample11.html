HTML Input form* Attribute

form* attribute for the HTML input element

  • form: equal to the id attribute of the form element it
  • formaction: attribute speicifes the URL of the file
  • formenctype: specifies how the form-data should be encoded (only post)
  • formmethod: defines the HTTP method for sending form-data to the action URL
  • formtarget: attribute specifies a name or a keyword that indicate where to display the response
    that is received after submitting the form.
  • formnovalidate: input element should not be validated when submutted.
